# VoiceMeal

Voice-based meal and calorie tracker for iOS.

## Tech Stack

- **SwiftUI** — Declarative UI framework
- **SwiftData** — Persistence and data modeling
- **Apple Speech Framework** — On-device speech recognition
- **Groq API (Llama 3)** — LLM-powered meal parsing and calorie estimation

## Sprint Plan

1. **Sprint 1 — Data Model & Basic UI:** Define SwiftData models (Meal, FoodItem) and build the main meal log screen.
2. **Sprint 2 — Speech Recognition:** Integrate Apple Speech framework for voice input to describe meals.
3. **Sprint 3 — Groq API Integration:** Send transcribed text to Groq (Llama 3) to parse food items and estimate calories.
4. **Sprint 4 — Dashboard & Insights:** Add daily/weekly calorie summary views and charts.
5. **Sprint 5 — Polish & Settings:** Error handling, onboarding, user preferences, and App Store preparation.
